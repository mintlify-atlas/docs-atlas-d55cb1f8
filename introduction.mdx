# Introduction to Docus

Welcome to **Docus**, a fully integrated documentation solution built with Nuxt UI and powered by the best of the Nuxt ecosystem.

## What is Docus?

Docus is a modern documentation framework that combines the power of [Nuxt 4](https://nuxt.com), [Nuxt Content](https://content.nuxt.com), and [Nuxt UI](https://ui.nuxt.com) to deliver beautiful, performant documentation sites. Built as a Nuxt layer, Docus provides a clean, minimal theme that lets you focus on what matters most: writing great content.

The framework uses enhanced Markdown with [MDC syntax](https://content.nuxt.com/docs/files/markdown#mdc-syntax), allowing you to embed Vue components directly in your documentation without writing any JavaScript.

## Built With

Your Docus project comes pre-configured with the best of the Nuxt ecosystem:

- **[Nuxt 4](https://nuxt.com)** - The Vue-based web framework
- **[Nuxt Content](https://content.nuxt.com/)** - File-based CMS with powerful querying
- **[Nuxt UI](https://ui.nuxt.com)** - Fully styled and customizable UI components
- **[Nuxt Image](https://image.nuxt.com/)** - Optimized image handling and processing
- **[Tailwind CSS 4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[Nuxt i18n](https://i18n.nuxtjs.org/)** - Internationalization support (optional)
